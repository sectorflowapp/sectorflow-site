What's inside

Three modules. Every flight.

Pilot tools for time calculations, crew rest planning, and in-flight prayer time support. Built for the pace of airline operations, with clear cockpit workflow and local calculations on iPhone.

Flight Tools

Crew Rest Planner

Set your flight configuration once — takeoff, landing, rest type — and SectorFlow helps calculate each rest block. This flight crew rest calculator keeps UTC timing readable so pilots can review rest plans quickly.

  • SSLL, SLLS, and standard rest configurations
  • Live countdown per block with compliance status
  • UTC-native — no local time confusion mid-flight
  • QR code sharing for seamless crew handoff
Time Tools

Time Calculator

An aviation time calculator for duration arithmetic, 24-hour clock calculations, and UTC offset conversion in one place. Handle pilot time calculator tasks in seconds, then move on to the next flight preparation step.

  • Duration, Clock, and Time Zone modes
  • Live UTC with local time at arrival
  • Fully offline — no internet dependency
Flight Tools

In-Flight Prayer Times

Accurate UTC prayer-time estimation at altitude. Enter your GPS position, route, or waypoint — SectorFlow handles the solar geometry. Field state persists between sessions so setup is a one-time task.

  • GPS, Route, and Waypoint input modes
  • Altitude-adjusted sunrise and sunset
  • Fajr, Sunrise, Dhuhr, Asr, Maghrib, Isha
  • All times displayed in UTC

Security isn't a feature.
It's the foundation.

We believe pilots deserve clear privacy practices around the tools they use. SectorFlow keeps app calculations local by design, while the website uses support and security services only where needed.

100% on-device processing

Every app calculation runs entirely on your iPhone. No server is needed for normal pilot tool use.

No app account required

No sign-up or profile is required for the iPhone app. Support form details are submitted only when you choose to contact us.

Clear privacy practices

The iPhone app is designed to avoid hidden sync and background uploads. The website may use Cloudflare for privacy-preserving analytics, security, and spam protection.
